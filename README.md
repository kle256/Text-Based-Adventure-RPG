# Text-Based Adventure RPG

A Python text-based RPG focused on object-oriented programming, inheritance-based class systems, state-driven gameplay flow, and turn-based combat.

## Project Status

Currently in development.

## Features

- Object-oriented adventurer and monster classes
- Inheritance-based class system
- Turn-based combat prototype
- Procedural character stat generation
- Randomized enemy encounters
- State machine structure for managing game flow

## Tech Stack

- Python
- Git / GitHub
- JSON for game data and save files

## Planned Features

- Multiple adventurer classes
- Monster types with unique behaviours
- Story and quest system
- Inventory and item system
- Save/load functionality
- Guild management mechanics

## What I’m Learning

- Object-oriented programming
- File organization across multiple Python modules
- State machines
- Data-driven design using JSON
- Version control with Git
